 テーマ「lp@site」2020/07/29版
====================================

------------------------------------
 インストールデータをインポートする
------------------------------------

1. このテーマはsiteの子ブログとして作られることを前提にデータを作っています。
	siteをインストールしたブログに子ブログを作ってください。

2. このテーマのインストールデータですが、新規インストーラーには用意していません。
       メンテナンス画面（https://ドメイン/〇〇（setupをリネームしたディレクトリ名）/index.php）よりインポートしてください。

3. インポートできたら、サイトを訪れてデータが入っているを確認してください。


------------------------------------
 テーマについて
------------------------------------
※このテーマは、siteテーマを継承したテーマになります。
 テーマの継承についてはこちら： テーマの継承 | テーマ | ドキュメント | a-blog cms developer
 https://developer.a-blogcms.jp/document/themes/inherit.html

------------------------------------
 テーマフォルダ直下のテンプレート
------------------------------------

■ entry.html
詳細ページのテンプレートです。

■ template.yaml
lp@siteテーマを選択したときに、テンプレートをまとめて設定できるファイルです。使用するかどうかは管理画面＞コンフィグ＞テーマ設定 にて変更できます。

■ thanks.html
ユニットで挿入できるフォームの終了画面です。

------------------------------------
フォルダ構成
------------------------------------
■ admin
lp@siteテーマオリジナルの管理画面のテンプレートが入ったフォルダ。
lp@siteテーマでは主に以下のテンプレートが入っています。
・ カスタムフィールド（エントリー）
・ 拡張ユニット

■ css
lp@siteテーマで使っているCSSが入ったフォルダです。

■ form
カスタムユニット「フォーム」で使用するフォルダです。

■ images
lp@siteテーマで使っている画像が入ったフォルダです

■ include
lp@siteテーマ内のインクルードしているテンプレートをまとめたフォルダです。

■ js
lp@siteテーマで使っているJavaScriptが入ったフォルダです。

■ scss
lp@siteテーマで使っているSCSSファイルが入っているフォルダです。
このscssフォルダに入っているSCSSファイルをコンパイルしてCSSを出力しています。
